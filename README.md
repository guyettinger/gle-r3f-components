# gle-r3f-components
A react-three-fiber component library

## Documentation
[Storybook](https://guyettinger.github.io/gle-r3f-components/)

## Installation
```shell
npm install gle-r3f-components@latest
```

## Development
Install
```
npm install
```
Build Library
```
npm run build
```
Run Tests
```
npm run test
```
Run Storybook
```
npm run storybook
```
